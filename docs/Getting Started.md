# Tổng quan
Đây là guide hướng dẫn sử dụng Claude Code với VS Code. 
Mục tiêu là chia sẻ một workflow đơn giản, dễ tiếp cận. Đồng thời tạo một nơi để mọi người chia sẻ với nhau các /skills bổ ích,

# Bắt đầu
## Environment: VS Code hoặc Cursor
- Cả VS Code và Cursor đều là những IDE phổ biến. Ta sẽ sử dụng chúng kèm với Terminal để gọi Claude Code

## Claude Code
- Claude Code là một coding tool, cho phép ta truy cập Claude models trực tiếp từ Terminal hoặc IDE hỗ trợ như VS Code và Cursor. 
- Để sử dụng, người dùng cần thực hiện 3 việc:
1. Có tài khoản Claude Pro, hoặc Max, hoặc sử dụng Claude Console API
2. Tải Claude Code
3. Đăng nhập vào Claude từ Terminal trong VS Code.

- Tài khoản đăng ký Claude Subscription cho chúng ta sử dụng 2 sản phẩm AI
1. Claude, sử dụng để viết, làm nghiên cứu, phân tích và hơn thế.
2. Claude Code, sử dụng để code với trợ giúp từ Claude trong terminal.



Chi tiết cách làm có thể refer đến link https://support.claude.com/en/articles/11145838-use-claude-code-with-your-pro-or-max-plan
