# Getting Started

How to install and sign in so you can use Claude Code with VS Code (or Cursor).

## Environment: VS Code or Cursor

- VS Code and Cursor are both popular IDEs. We'll use one of them together with the Terminal to run Claude Code.

## Claude Code

- Claude Code is a coding tool that lets us access Claude models straight from the Terminal, or from a supported IDE like VS Code or Cursor.
- To use it, we need three things:
    1. A Claude Pro or Max account, or a Claude Console API key.
    2. Claude Code installed.
    3. To sign in to Claude from the Terminal inside VS Code.

A Claude subscription gives us two AI products:

1. **Claude** — for writing, research, analysis, and more.
2. **Claude Code** — for coding with Claude's help in the terminal.

For the details, see [Use Claude Code with your Pro or Max plan](https://support.claude.com/en/articles/11145838-use-claude-code-with-your-pro-or-max-plan).

## Installation

Follow the official [Claude Code quickstart](https://code.claude.com/docs/en/quickstart).
